The Data IDE for Developers
  • Query all your data in one place: SQL, HTTP, files, etc.
  • Script against data in Python, JavaScript
  • Create visualizations
  • Open source

Online Environment

An online environment is available for data exploration in the browser. In the online environment you have access to the majority of DataStation's features limited only by the fact that the entire application runs without a server.

Entirely in-memory

The online environment runs entirely without a server, entirely in your browser. You have access to one project and it persists to localstorage.

Own your data

Since there is no server, you can safely use it for work or home. The data never leaves your computer.

Share projects safely

Project state can be serialized to a URL by clicking the Share mbutton in the top right. The data in the URL not stored in any database (though it will be stored in DataStation web server access logs if someone uses the URL).

Make HTTP requests

You can make HTTP requests to any server that set CORS headers. However, you cannot make requests to SQL databases since these requests use a protocol other than HTTP.

In-memory SQL

You do have access to an in-memory SQL implementation so you can still perform SQL queries against any other data in the online environment.

Script in Python and JavaScript

The online environment supports JavaScript and Python today, limited to languages that have been implemented in JavaScript already.

Great for desktop

The online environment is for use on desktop computers, so you'll have to try it out on another device.
