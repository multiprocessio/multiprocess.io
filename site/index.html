In-memory App

The in-memory app is freely available for data exploration in the browser. No account required!

Entirely in-memory

HTTP requests happen from the browser without any proxy (so your server must set CORS headers). SQL, JavaScript, and Python are available and interpreted within the browser.

Own your data

Since there is no server, you can safely use it at work or home. The data never leaves your computer.

Share projects safely

Project state can be serialized to a shareable URL by clicking the Share button in the top right.

Great for desktop

The in-memory app is for use on desktop computers, so you'll have to try it out on another device.
