The Data IDE for Developers
  • Query all your data in one place: SQL, HTTP, files, etc.
  • Script against data in Python, JavaScript
  • Create visualizations
  • Open source

Online Environment

An online environment is available for data exploration in the browser. It has access to the majority of DataStation's features limited only by the fact that the entire application runs without any server.

Entirely in-memory

HTTP requests happen from the browser without any proxy (so your server must set CORS headers). SQL, JavaScript, and Python are available and interpreted within the browser.

Own your data

Since there is no server, you can safely use it at work or home. The data never leaves your computer.

Share projects safely

Project state can be serialized to a URL by clicking the Share button in the top right. The data in the URL is not stored in any database (though it will be stored in DataStation web server access logs if someone uses the URL).

Great for desktop

The online environment is for use on desktop computers, so you'll have to try it out on another device.
