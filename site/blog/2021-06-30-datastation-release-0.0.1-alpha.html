I introduced DataStation in the first blog post. It is an IDE for manipulating data (from HTTP servers, SQL databases, local files, and so on) via scripting languages and visualizing the result -- all in a single application.

You can now try out DataStation on Mac and Windows. Builds for Linux and non-x86 architectures (ARM, basically) will follow shortly. If you'd prefer to try it out before installing it, you can still do so via the online environment.

DataStation remains an early-stage alpha. Do not use it for any critical data unless you are backing up the .dsproj files it saves in ~/DataStationProjects/. Bug reports are extremely welcome when accompanied by details for reproduction. Our Discord is open.

I'm looking forward to hearing from you.
