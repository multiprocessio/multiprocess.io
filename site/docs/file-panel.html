File panels allow you to read files from disk. Every time the panel is run the file is read again (except for in the online environment).

With Server

You can read files from remote servers with an SSH tunnel by checking the With Server option. Once selected, you will be prompted to pick from one of the SSH Connections you set up previously.

Online Environment Caveats

In the online environment the browser won't allow us to remember the file you've selected or read it again without you reselecting it. So it is only read once when you pick a file.

Re-running the panel will not cause it to be re-read. And reloading the page will cause it to be forgotten until you select the file again.

You cannot read files over SSH in this mode.
