File Panels

These panels allow you to import files into DataStation from disk. Every time the panel is run the file is read again.

Supported file formats

Data

  • JSON
  • CSV
  • Parquet
  • XLSX

Logs

  • Apache2 Access Logs
  • Apache2 Error Logs
  • Nginx Access Logs
  • Newline-delimited JSON
  • Newline-delimited Custom Regex

With Server

You can read files from remote servers with an SSH tunnel by checking the With Server option. Once selected, you will be prompted to pick from one of the SSH connections you set up previously.
