Reference

DataStation is a data IDE for developers. It allows you to easily build graphs and tables with data pulled from all kinds of databases, HTTP servers, and files (CSV, Parquet, Excel, nginx access logs, syslog, etc.). Need to join or munge data? Write embedded scripts as needed in Python, JavaScript, Ruby, R, or Julia. All in one application.
