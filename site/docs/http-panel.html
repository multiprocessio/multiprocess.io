HTTP panels are for making HTTP requests. You can make GET, POST, PUT, HEAD, or DELETE requests.

With Server

HTTP requests can be proxied through an SSH tunnel by checking the With Server option. Once selected, you will be prompted to pick from one of the SSH Connections you set up previously.

Online Environment Caveats

In the online environment you can only make requests to servers that set CORS headers since there is no server.

You cannot proxy HTTP requests through an SSH tunnel in this mode.
